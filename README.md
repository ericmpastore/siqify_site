# siqify_site
Future repository for the Siqify Analytics website.

## Site Structure:
Home - Home page for the site, with an overview of services and current projects.
About - Learn about the company and see instructor bios. Includes downloadable resumes.
Courses - Learn about the course offerings from Siqify. Includes downloadable syllabus for each course.
Contact - Request quotes for training or ask general questions.
